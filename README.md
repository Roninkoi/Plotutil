# Plot utilities in Fortran

Uses Gnuplot to draw plots from data files. Data columns separated by spaces or tabs, # is comment.

Usage e.g.

```
./run.sh gnuplot.plt data.txt
```

Contains:
- array handling utils
- sorting
- line fitting
